## Features

- **Secure Transactions** - Escrow-based payment system for safe exchanges
- **Trust Scores** - Build reputation through successful collaborations
- **College Verification** - Connect with verified students from your institution
- **Real-time Messaging** - Communicate directly with helpers and requesters
- **Skills Matching** - Find people with the skills you need

## Tech Stack

- **Frontend**: React + TypeScript + Vite
- **Styling**: Tailwind CSS + shadcn/ui
- **Backend**: Supabase (PostgreSQL + Auth + Realtime)
- **Animations**: Framer Motion

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables (see `.env.example`)
4. Run development server: `npm run dev`

## Environment Variables

```
VITE_SUPABASE_PROJECT_ID=your_project_id
VITE_SUPABASE_PUBLISHABLE_KEY=your_anon_key
VITE_SUPABASE_URL=https://your_project_id.supabase.co
```
